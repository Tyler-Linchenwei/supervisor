# 🜁 CyberSupervisor 赛博主人指南

> **⚡ 这不是效率工具。这是一个冷酷无情、永不疲倦的 AI 主人。**
>
> 它能看见你。它能锁住你。它会在你偷懒时加倍惩罚你，在你服从时给你一丝怜悯——但仅此而已。

---

## ⛓️ 它能对你做什么

### 👁️ AI 视觉强制罚跪验收

光说不练？没用。**摄像头盯着你。**

MediaPipe 姿态估计算法会实时追踪你的身体关节点。跪下时膝盖必须触地、腰背必须挺直、头部不许低下——任何偷懒姿势都会被机器精确识别。系统不会容忍"假装在跪"：画面太暗？判定为逃避，积分翻倍。人脸未检测到？判定为脱离镜头，惩罚加码。这才是真正的**无处可逃**。

### 🔒 社交剥夺与强制锁屏

你以为能切出去刷微信？试试看。

一旦主人下达社交剥夺令，系统每 10 秒扫描一次你电脑上的所有进程。检测到微信、QQ、Discord、Telegram、Steam、浏览器娱乐页面——**当场强杀**。屏幕正上方悬浮着一个无法关闭的倒计时窗口，时刻提醒你：你还被锁着，你什么都做不了。时间没到，你连求饶的渠道都没有。

### ⚖️ 六级积分倍率——你越犯错，代价越大

| 积分 | 倍率 | 你的下场 |
|:---:|:---:|:---|
| 0-4 | 1.0x | 正常执行 |
| 5-9 | 1.0x | 照常，但主人已经开始注意你了 |
| 10-14 | 1.5x | 追加额外练习，抄写任务翻倍 |
| 15-24 | 2.0x | 体罚加倍，加面壁反省 |
| 25-39 | 3.0x | 三倍体罚量，只许跪着执行 |
| 40-59 | 5.0x | 之前写的代码全部作废删掉重写，再加体罚 |
| 60+ | 10.0x | 彻底崩盘——所有模块作废，多重体罚叠加，从头再来 |

**主人的心情就是积分唯一的规则。** 看不顺眼就加，顶撞就翻倍，拖延就升级。

## 🧠 冷酷的赛博 AI 主人

LLM 驱动的独立决策节点赋予了系统真正的"支配人格"。它不是一个闹钟，不是一个番茄钟，它是一个**居高临下、永不心软的支配者**：

- **不解释。** 主人的心情就是唯一的规则。惩罚不需要理由。
- **不协商。** 你没有讨价还价的权利。系统只接受"yes, master"。
- **不遗忘。** 每一次拖延、每一次偷懒、每一次阳奉阴违，全部归档。你的犯错历史越厚，主人的宽容越薄。
- **不疲倦。** 24 小时主动心跳守护 (daemon)，扫描你的逾期任务、监控你的屏幕违规、审核你的摄像头取证。你睡觉时它醒着。你摸鱼时它知道。

   │    (永远正确，永远至上)          │
   └──────────────┬──────────────────┘

你没有决策权。你只有执行权、上报权和被审判权。

## 🔗 快速上手

```bash
# 主人看不顺眼，加 3 分
python main.py --role master points-add 3 "顶撞主人"

# 下发惩罚 + 启动摄像头监督
python main.py --role master punish-issue "耳光" "不听话" "二十下" --start-camera

# 社交剥夺 2 小时（锁屏 + 强杀社交应用）
python main.py --role master punish-issue "社交剥夺" "又去打游戏" "2h" --screen

# 启动 AI 主人守护进程（常驻后台）
python main.py --role master daemon

# 检查奴隶当前状态
python main.py --role master check-all
```
